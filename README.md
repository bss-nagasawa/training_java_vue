# BSS 研修課題リポジトリ

この課題はバックエンドを Java で構築します。

## 管理者インタフェース

### 1. 登録インタフェース

#### 1-1. 基本情報

- **エンドポイント**: `/admin/register`
- **HTTPメソッド**: `POST`
- **メソッド名**: `register`
- **インターフェースの説明**: このインターフェースは、管理者を登録するために使用されます。

#### 1-2. リクエストパラメータ

| パラメータ名 | 説明       | 種類    |
|--------------|------------|---------|
| register     | 登録者名   | string  |
| password     | パスワード | Integer |
| auth         | 権限       | Integer |

#### 1-3. レスポンスデータ

- **レスポンスデータの種類**: `json`
- **レスポンスデータ説明**:

| レスポンスデータ名 | 種類    | 備考         |
|-------------------|---------|--------------|
| code              | number  | 0-成功、1-失敗 |
| message           | string  | メッセージ    |
| data              | object  | 戻り値        |
